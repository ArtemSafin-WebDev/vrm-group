Описание

Сварочное решение KUKA KR 120 QUANTEC предназначено для замены ручного труда на роботизированный. Наиболее востребовано в отрасли машиностроения.

Алгоритм настройки робота следующий: оператор создает программу траектории перемещения, назначает сварочные параметры и сохраняет. Далее робот выполняет последовательные операции по заданной программе требуемое количество раз. При переходе к новому изделию последовательность повторяется.
